Nafn Formúla

  • Til þess að flytja rafmagn á milli staða eru notaðar rafleiðslur / rafleiðarar
  • Kostur er að leiðarar hafi sem minnst viðnám of því eru þeir úr efnum sem hafa : lágt eðlisviðnám = p og háa eðlisleiðni = G
  • Við hönnum raflagna ig rafrása þarf að hafa eftirtalin fjögur atriði til viðmiðunar:
    • Viðnámið í leiðurunum = R1
    • Heildarviðnám rásarinnar = Rh
    • Strauminn = I
    • Spennuföllin = U
      • Stafurinn "I" í R1 merkir viðnámið í leiðaranum og "h" fyir R merkir heildarviðnám rásarinnar
  • Eins og áður hefur komið fram hafa leiðarar alltaf eitthvert viðnám og fer það eftir efni, lengd og gildleika leiðarans
  • Leiðarar sem flytja straum til einhvers ákveðins álags eru a.m.k tveir og eru alltaf jafn sverir og jafn langir
  • Þeir hafa því jafn mikið viðnám
  • Hluti spennunnar fer í það að knýja strauminn gegnum leiðarana
  • Þessi hluti spennunnar er oft nefndur spennufall í leiðurum
  • Þegar viðnám í leiðara er reiknað út frá upplýsingum um efni, lengd og þverflatarmál hans er notuð líkingin:
    • R1 = Viðnámið í leiðarana
    • p = Eðlisviðnám
    • l = lengd
    • A = Þverflatarmál mm^2
    • Ø = Þvermál
      • R1 = (p * l) / A
  • Hvernig er þverflatarmál reiknað? :
    • A = r² * π
    • A = Þverflatarmál / kvaðrat
    • R = Radíus
    • π = 3,14
  • Heildarviðnám straumrásar er samanlagt viðnám leiðara og álags
    • Rh = heildarviðnám í rásinni
    • R1 = viðnámið í leiðara
    • R = Viðnám álgasins (tækisins)
    • Rh = R1 + R
  • Spennutap í leiðara er táknað með ΔU
    • ΔU = I * R1
      eða
      ΔU = I * (p * l) / A
      vegna þess að
      R1 = (p * l) / A
